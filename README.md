============

My personal website's repository.

## Tooling

```sh
$ # Install all development tools
$ npm install
$ # Watch For Changes & Automatically Refresh Across Devices
$ npm start
$ # Build & Optimize
$ npm run build
$ # Run tests & linting
$ npm test
```
